# Student Management System

This project is the initial scaffold for a MERN stack Student Management System.

## Project Structure

- client: React frontend built with Vite.
- server: Node.js and Express backend.
- server/config: Configuration files such as environment settings and database connection setup.
- server/controllers: Request handlers for application logic.
- server/middleware: Custom middleware for validation, authentication, or error handling.
- server/models: Mongoose models for data schemas.
- server/routes: API route definitions.

## Getting Started

### Frontend

```bash
cd client
npm run dev
```

### Backend

```bash
cd server
node index.js
```

## Notes

CRUD functionality has not been implemented yet. This scaffold is intended as a starting point for building the full application.
